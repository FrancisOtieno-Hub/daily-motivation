# Daily Motivation Web App
A simple, beautiful, and interactive **Daily Motivation web app** built with HTML, CSS, and JavaScript. It displays daily motivational quotes, allows you to generate random quotes, share them, and optionally receive daily notifications.
This project can also be hosted as a **Progressive Web App (PWA)** or wrapped into an APK for Android devices.

## **Features**
* Display a **daily motivational quote** based on the day of the year.
* **Random quote generator** at the click of a button.
* **Share quotes** via native sharing (or copy to clipboard on unsupported browsers).
* **Enable daily notifications** (9 AM local time) for a motivational boost.
* **Splash screen** on load for a polished, app-like experience.
* Fully **responsive** design suitable for mobile and desktop.
* Easily converted to a **PWA** or wrapped into an APK.

## **Installation / Usage**
1. **Clone the repository**
```bash
git clone https://github.com/FrancisOtieno-Hub/daily-motivation.git
```
2. **Open `index.html`** in any modern browser.
   * No backend or server required.
3. **Optional:** Host online for mobile access or PWA functionality.
   * Use [Vercel](https://vercel.com), [Netlify](https://www.netlify.com/), or similar platforms.

## **Usage Instructions**
* **Random Quote**: Click “Random Quote” to get a new motivational quote instantly.
* **Share**: Click “Share” to copy the quote or open the native share menu.
* **Enable Notifications**: Click to allow daily notifications at 9 AM.
